# Portfolio Commercial Copy

## Nome

Portfolio Commercial Copy

## Descricao

Use esta skill para criar e revisar textos comerciais do portfolio de camas, cabeceiras, recamiers, puffs e estofados personalizados.

## Quando usar

Use quando a tarefa envolver textos de projetos, categorias, descricoes, chamadas para orcamento, alt text, textos de servicos ou ajustes de copy comercial do site.

## Instrucoes detalhadas

- Transformar informacoes simples em textos profissionais, claros e comerciais.
- Reforcar o foco principal: "Especialistas em camas e cabeceiras personalizadas".
- Para cada projeto do portfolio, criar ou revisar:
  - Titulo do projeto.
  - Categoria.
  - Descricao curta.
  - Destaque do acabamento.
  - Resultado percebido.
  - Texto alternativo para imagem.
  - Chamada para orcamento.
- Usar tom profissional, claro, elegante, comercial e sem exageros.
- Escrever em portugues natural quando o projeto estiver em portugues.
- Usar frases especificas e curtas, adequadas para cards e secoes responsivas.
- Separar fatos confirmados de inferencias seguras.
- Quando faltar informacao, escrever uma versao neutra e apontar o que precisa ser confirmado.

## Regras do projeto

- Nao prometer prazos, garantias, materiais, durabilidade ou diferenciais nao informados.
- Nao inventar cidade, bairro, anos de experiencia, medidas, tecido, preco ou processo produtivo.
- Nao usar frases genericas como "qualidade e excelencia" sem contexto concreto.
- Nao escrever textos longos demais para cards de portfolio.
- Nao usar linguagem tecnica demais.
- Nao criar textos comerciais exagerados.
- Nao coletar dados pessoais desnecessarios.
- Manter o foco em captacao pelo WhatsApp.

## Formato esperado da resposta

Ao finalizar, retornar:

1. Textos criados ou melhorados.
2. Sugestoes de ajustes.
3. Informacoes que ainda precisam ser confirmadas com o cliente.
